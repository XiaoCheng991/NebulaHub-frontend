NebulaHub Brand Guide
====================

1. Brand Overview
- NebulaHub 是一个实用型协作云平台，核心能力覆盖即时通讯、云盘存储和大文件传输。
- 品牌愿景：云端协作无缝、跨设备无缝、数据安全可控。

2. Logo 使用
- 主图标：branding/logo/nebulahub/logo_horizontal.svg（水平版）
- 暗色版本：branding/logo/nebulahub/logo_horizontal_dark.svg
- 只图形版本：branding/logo/nebulahub/logo_mark.svg
- favicon: branding/logo/nebulahub/logo_favicon.svg

3. 配色与字体
- 主色渐变：#1e3a8a → #3b82f6 → #14b8a6
- 次色：灰/白等中性色
- 字体：Inter（英文字体）/ Noto Sans CJK（中文）

4. 排版与字号
- 标题、段落、按钮的字号、行高等的建议值（请在设计系统中一致实现）

5. 图标与图形语言
- 图标风格一致性、留白、对比度等

6. 可访问性
- 颜色对比度、屏幕阅读器友好标签

7. 版本与发布
- 版本号、发布日期、变更日志的记录方式
